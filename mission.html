1장 HTML 이란

HTML(Hyper Text Markup Language)


프로그래밍 언어로 웹페이지를 만들 때 사용
Hyper Text = 링크 / Markup Language = 태그를 이용한 언어
확장자가 html


2장 HTML태그

태그


html은 태그들의 집합이며
태그는 무언가를 표시하기 위한 꼬리표, 이름표 라는 의미가 있으며
html에서도 이와 비슷한 의미로 해석함

    <h1>hello, html</h1>

  1. h1: 태그 이름
  2. <h1>: 시작태그
  3. </h1>: 시작태그
  4. hello HTML: 내용
  5. <h1>hello, html</h1>: 요소

속성

태그에 추가로 정보를 제공 태그의 동작, 표현을 제어할 수 있는 설정값을 의미
선언 : 시작 태그 안에 <태그명 속성명="값">으로 구성
(이름, 값 사이에 = 기호를 제외한 어떠한 공백도 들어가면 안 됨)
여러 속성을 선언할 경우 공백으로 구분
선언 순서는 중요하지 않음

ex) <h1 id="title">hello, html</h1>

  • id: 속성
  • "title" 속성 값
